<h3>Opção de Trabalho</h3>
Opção 2 – algoritmo de transformação AFe -> AFN.

<h3>Descrição do Projeto</h3>
Este trabalho tem como objetivo implementar um algoritmo capaz de transformar um autômato finito com transições vazias (AFe) em um autômato finito não determinístico (AFN)

<h3>Requisitos Atendidos</h3>
Representação dos estados do autômato<br>
Representação das transições<br>
Representação do alfabeto<br>
Suporte ao símbolo vazio<br>
Limite máximo de 3 símbolos no alfabeto<br>
Remoção das transições epsilon no AFN convertido<br>
Geração das novas transições do AFN<br>
Atualização dos estados finais no AFN<br>
Validação da palavra lida<br>
Impressão da palavra aceita<br>
Aplicação prática de conceitos de Linguagens Formais e Autômatos<br>
